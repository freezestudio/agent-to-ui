# 规范 v1.0

此目录包含 A2UI 1.0 版的规范。先前在草案阶段称为 0.10 版。

**此规范目前是成为稳定版的候选版本。**

一旦足够数量的渲染器已被移植使用此版本的规范，并且该过程的反馈已被整合，这将成为规范的稳定 1.0 发布版。接受候选规范的破坏性更改有很高的门槛。

如果你有提议的更改或新功能，请向 [A2UI 仓库](https://github.com/a2ui-project/a2ui) 提交 issue 或拉取请求。
