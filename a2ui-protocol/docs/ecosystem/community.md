# 社区

欢迎来到 A2UI 社区！项目正在共同构建代理驱动界面的未来。

## 参与

A2UI 是一个开源项目，采用 Apache 2.0 许可。欢迎开发者、研究人员和任何对推进代理用户界面感兴趣的人贡献。

## 社区展示

用 A2UI 构建了什么？展示你的作品！

分享作品的最佳方式是在 [GitHub Discussions](https://github.com/a2ui-project/a2ui/discussions) 中**发帖**，附上简短描述、代码链接和演示视频（4 分钟以内）。值得注意的项目会出现在文档和发布说明中。

**社区构建的渲染器**列在[生态系统渲染器](renderers.md)页面——在那里查看社区实现以及如何添加自己的渲染器的说明。

## 项目合作伙伴

A2UI 与 Google Opal、Flutter（GenUI SDK）、Gemini Enterprise、AG-UI / CopilotKit 和 A2A 团队合作开发。有关完整详情和案例研究，请参见 [A2UI 在现实中的应用](a2ui-in-the-world.md)。

如果你对 A2UI 做出了重大贡献，请将你的名字提交到此列表。

## 行为准则

A2UI 致力于为每个人提供欢迎和包容的环境。所有参与者都应：

- 尊重和体贴他人。
- 欢迎新加入者并帮助他们上手。
- 关注对社区最有利的事情。
- 对他人表现出同理心。

不可接受的行为将不被容忍。如有问题请向项目维护者报告。

## 认可

所有贡献都受到赞赏！贡献者在以下地方被认可：

- GitHub 的贡献者图。
- 发布说明。
- 项目致谢。

重要贡献者可能被邀请加入项目维护者团队。

## 保持更新

要了解项目更新：

- **关注 GitHub 仓库**以获取通知。
- **加星标**以收藏和表示支持。
- **关注发布**以获取新版本通知。

## 贡献方式

**[github.com/a2ui-project/a2ui](https://github.com/a2ui-project/a2ui)**

- **报告问题**：发现 bug？[开启一个 issue](https://github.com/a2ui-project/a2ui/issues)。
- **构建渲染器**：参见[路线图](../roadmap.md)了解计划的框架。
- **分享示例**：在 X/LinkedIn 上使用 `#A2UI` 发帖，在 [GitHub Discussions](https://github.com/a2ui-project/a2ui/discussions) 中开启讨论。
- **改进文档**：欢迎在 `docs/` 目录提交 PR。
